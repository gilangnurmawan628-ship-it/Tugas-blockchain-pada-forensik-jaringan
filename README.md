Deskripsi Tugas

Tugas ini bertujuan untuk melakukan akuisisi bukti jaringan menggunakan file PCAP, menghitung hash SHA-256 sebagai identitas digital bukti, serta mengimplementasikan blockchain sederhana untuk menjaga integritas dan chain of custody bukti digital.

Tools yang digunakan 
< Wireshark

Cara Menjalankan Program
Pastikan Python 3 sudah terinstal.
Simpan file PCAP pada folder evidence.
Jalankan program:
python blockchain_pcap.py
Hasil SHA-256

Program menghasilkan nilai hash SHA-256 untuk setiap file PCAP.

Hasil Blockchain

Blockchain dibuat menggunakan:

Genesis Block
5 Evidence Block
Previous Hash
Block Hash
Hasil Validasi Blockchain

Contoh output:

Blockchain Validation : VALID
